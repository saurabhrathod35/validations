## Validations-lib
_this lib contains all type of validation in node and angular 6_

## How to use

You have to pass this object 
```
var object = {
     title: 'number 1';
     type: 'text';
     condition: 'gt';
     currentValue: '1';
     message: 'number 1 must be greater than 25 ';
     params: [25];
     uid:'#1';
}
```
Your output will be 
```
{'result':false,message:'number 1 must be greater than 25'}
```
